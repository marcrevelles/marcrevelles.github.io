# Revelles Lab
This is my technical blog, where I post about cybersecurity, self-hosting, and hardware hacking. The main objective of the blog is to document the challenges that I face while building secure and reliable infrastructure. I hope that you find the information on the blog to be helpful.

👉 [revelleslab.com](https://revelleslab.com)


## Tech stack
The blog is built using:
- Jekyll static site generator
- Chirpy theme
- GitHub Pages for hosting
- GitHub Actions for CI/CD
